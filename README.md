# Trivia Quiz App 💡
An elegant, trivia application where enthusiasts can test their knowledge. This project focuses on design starting from a Figma wireframe to a fully functional React SPA.

## Tech Stack
**Framework:** React (Vite)
**Styling:** CSS & HTML
**Design:** Figma
**Back-end:** JSON Server (Mock API)

## Features
- Interactive quiz interface with instant feedback
- Data persistence usung a loal `db.json` file.
- Responsive layout for mobile and desktop.

## Set-Up Instructions
    Ensure you have [Node.js] (https://nodejs.org) installed.


1. **Clone and Install:**
    ```bash
    git clone (https://github.com/lennymethu-sys/trivia-quiz-app.git)
    cd trivia-quiz-app
    npm install

2. **Run the Database:**
    (Ensure you have json-server installed or use npx)

    npm json-server --watch db.json --port 3001

3. **Launch the App**

    npm run dev
